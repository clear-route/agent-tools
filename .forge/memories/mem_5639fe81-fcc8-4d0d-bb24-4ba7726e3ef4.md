For Forge custom tools in `clear-route/agent-tools`, agent-facing documentation should be placed in a `README.md` file inside the tool's folder. This README serves as the agent's context/methodology guide when using the tool — since Forge does not read SKILL.md files the way Claude Code does. The `excalidraw-assistant` README contains the full design methodology (207 lines) covering core philosophy, visual patterns, and the render-validate-fix loop.
