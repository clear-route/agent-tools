For `clear-route/agent-tools`, compiled Go binaries and Python `.venv` directories must NOT be committed to the repo. The repo should contain only source files. Compiled artifacts belong in `~/.forge/tools/<tool-name>/` after a build/install step. Reference: `excalidraw-assistant` had a 2.9 MB binary and 112 MB `.venv` accidentally copied in and had to be cleaned out.
