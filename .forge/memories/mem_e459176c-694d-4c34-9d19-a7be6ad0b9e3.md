Build/install pattern for `clear-route/agent-tools` Go tools: source lives in `~/Agents/agent-tools/<tool-name>/`; build with `go build -o ~/.forge/tools/<tool-name>/<tool-name> <tool-name>.go` from the tool's source directory. For excalidraw-assistant, the Python renderer also requires a one-time setup: `cd ~/.forge/tools/excalidraw-assistant/references && uv sync && uv run playwright install chromium`.
