The `excalidraw-assistant` tool architecture has been redesigned: the agent (LLM) is responsible for generating the Excalidraw JSON using its own capabilities (informed by reading reference files), and the Go tool acts as a **rendering and validation engine** only. The tool accepts pre-generated JSON, validates it against the schema, and renders it to PNG — providing feedback errors for the agent to iterate on. The tool does NOT generate diagram content itself.
