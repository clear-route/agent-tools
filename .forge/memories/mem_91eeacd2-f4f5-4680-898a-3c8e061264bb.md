The `excalidraw-assistant` tool architecture (finalized): the agent (LLM) generates Excalidraw JSON by reading reference files at `~/.forge/tools/excalidraw-assistant/references/` (`color-palette.md`, `element-templates.md`, `json-schema.md`), writes the JSON to a file, then calls the tool with `--input <path>`. The Go tool validates the JSON and renders it to PNG via the Python/Playwright pipeline. The tool does NOT generate diagram content. No color palette validation is done tool-side — the agent is trusted to read and follow `color-palette.md`, and users may explicitly override colors. Tool flags: `--input` (required), `--output` (optional, defaults to same name as input with `.png`), `--validate-only`, `--scale`, `--width`.
