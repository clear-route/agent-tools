The `excalidraw-assistant` tool in `clear-route/agent-tools` is implemented in Go (not as a Claude Code skill). The tool should include a `README.md` in its tools folder that agents can read to get context on how to use the tool — since Forge does not read SKILL.md files the same way Claude Code does. This README serves as the agent-facing documentation for the tool's capabilities and usage patterns.
