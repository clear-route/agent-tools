The `excalidraw-assistant` README.md in `clear-route/agent-tools` should NOT encourage agents to run shell commands. Instead, it should tell agents that the `references/` folder is located inside the custom tools installation directory (e.g. `~/.forge/tools/excalidraw-assistant/references/`) and that agents should read those reference files directly from that path to understand color palettes, element templates, and JSON schema during diagram construction.
